    # Moment.js库 vs Day.js库

    • Moment库，官网的描述

      • Moment 是一个 JavaScript 库，可以帮助我们快速处理时间和日期，已在数百万的项目中使用
      • Monent 对浏览器的兼容性比较好，在Internet Explorer 8+版本运行良好。
      • 现在比较多人反对使用 Moment是因为它的包大小。Moment 不适用于“tree-shaking”算法，因此往往会增加 Web 应用程序包的大小。如果需要国际化或时区支持，Moment 可以变得相当大。
      • Moment团队也希望我们在未来的新项目中不要使用Moment 。而推荐使用其它的替代品。例如：Day.js。

    • Day.js库，官网的描述

      • Day.js 是 Moment的缩小版。Day.js 拥有与 Moment相同的 API，并将其文件大小减少了 97%
      • Moment完整压缩文件的大小为 67+Kb，Day.js 压缩文件只有 2Kb
      • Day.js所有的 API 操作都将返回一个新的 Day.js 对象，这种设计能避免 bug 产生，减少调试时间。
      • Day.js 对国际化支持良好。国际化需手动加载，多国语言默认是不会被打包到Day.js中的。

    # Day.js 获取、设置、操作时间

    • 获取 （get）+ 设置 (set)
      • .year() 、.month() 、date() 获取年、月、日
      • .hour()、.minute()、.second() 获取时、分、秒
      • .day() 获取星期几
      • .format() 格式化日期
    • 操作日期和时间
      • .add(numbers , unit) - 添加时间
      • .subtract(numbers , unit) - 减去时间
      • .startOf(unit) - 时间的开始 (例如：获取今年的第一天零时零分零秒)
    let day = dayjs()

console.log(day.year()); // 2022 年
console.log(day.month()+1);; // 12  月
console.log(day.date());  // 15 日
console.log(day.hour()); // 15 小时
console.log(day.minute()); // 5 分钟
console.log(day.second()); // 53 秒
console.log(day.day());  // 4 星期

console.log(day.format()); // 2022-12-15T14:56:29+08:00
console.log(day.format("YYYY-MM-DD HH:mm:ss")); //2022-12-15 14:43:16

    # Day.js 解析 、国际化、插件

    • 解析时间

      • dayjs(毫秒|秒) - 时间戳（毫秒 和 秒）
      • dayjs('2022-06-15') - ISO 8601格式的字符串
      • dayjs(new Date()) - 接收日期对象
      // 1. 解析一个字符串(ISO 8601)类型的时间
let day1 = dayjs("2021-2-2 12:00:10");
console.log(day1.format("YYYY-MM-DD hh:mm:ss")); // 2021-02-02 12:00:10

// 2.解析时间戳(毫秒)
let day2 =  dayjs(1656206934331)
console.log(day2.format("YYYY-MM-DD hh:mm:ss"));  // 2022-06-26 09:28:54

// 3.解析时间戳(秒)
let day3 = dayjs.unix(1656206934)
console.log(day3.format("YYYY-MM-DD hh:mm:ss"));  // 2022-06-26 09:28:54

// 4.解析Date对象
let day4 = dayjs(new Date("2022-12-15"))
console.log(day4.format("YYYY-MM-DD hh:mm:ss"));   // 2022-12-15 08:00:00

    • Day.js 的插件应用

      • . fromNow() - 从现在开始的时间 （需要依赖：relativeTime 插件）

      • relativeTime 插件：

      • https://cdn.jsdelivr.net/npm/dayjs@1.11.3/plugin/relativeTime.js

      <script src="./libs/dayjs.js"></script>
<script src="https://cdn.jsdelivr.net/npm/dayjs@1.11.3/plugin/relativeTime.js"></script>
<script src="./libs/dayjs.zh-cn.min.js"></script>

// 1. 安装插件
dayjs.extend(dayjs_plugin_relativeTime)

// 2. 切换使用中文
dayjs.locale("zh-cn")

let day = dayjs(1656206934331)  // dayjs 对象

console.log( day.fromNow());   // 6 个月前
