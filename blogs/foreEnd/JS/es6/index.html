    # 1.ES6对象方法

    # 1.1 Object.is()

    Object.is() 方法用于确定两个值是否相同；它接受两个参数，这两个参数是要比较的值，并返回一个布尔值，指示两个参数是否相同

    console.log(Object.is(1,'1')); // false
    1

    • Object.is() 与 == 不同， == 运算符在判断相等前对两边的变量（如果它们不是同一类型）进行强制转换，而 Object.is 不会强制转换两边的值。

    • Object.is() 与 === 不同 ，差别是它们对待有符号的零和 NaN 不同

      • console.log(Number.NaN===NaN); // false
console.log(-0===0); // true

console.log( Object.is(Number.NaN,NaN)); // true
console.log(Object.is(-0,0)); // false

    # 1.2 Object.assgin()

    用于对象的合并，将源对象的所有可枚举数据，复制到目标对象

    Object.assign(target,sources)
    1
    • target 目标对象，接收源对象属性的对象，也是修改后的返回值
    • sources 源对象，包含将被合并的属性
    const target = {a:1,b:2};
const source = {b:4,c:5}

console.log( Object.assign(target,source));  // {a: 1, b: 4, c: 5}
    1
    2
    3
    4

    如果目标对象与源对象具有相同的 [key]，则目标对象中的属性将被源对象中的属性覆盖，后面的源对象的属性将类似地覆盖前面的源对象的属性。

    # 1.3 Object.keys()

    返回一个数组，成员时参数对象自身的所有可遍历属性的键名

    const target = {a:1,b:2};
console.log(Object.keys(target)); // ['a', 'b']
    1
    2

    # 1.4 Object.values()

    返回一个数组，成员是参数对象自身的所有可遍历属性的键值

    var obj = { foo: 'bar', baz: 42 };
console.log(Object.values(obj));  // ['bar', 42]
    1
    2

    # 1.5 Object.entries()

    Object.entries()返回一个数组，其元素是与直接在object上找到的可枚举属性键值对相对应的数组。

    const target = {a:1,b:2};
console.log(Object.entries(target));  //  [ ['a', 1], ['b', 2] ]
