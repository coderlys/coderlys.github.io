    常用数组方法

    9/16/2022 JS

    # 一：ES5及之前

    # concat()

    concat() 方法用于合并两个或多个数组，此方法不会更改现有数组，而是返回一个新的数组

    const arr1 = [1,2,3,4]
const arr2 = [4,5,2]
const arr3 = [6,6]
const newArr = arr1.concat(arr2).concat(arr3)
console.log(newArr);  // [1, 2, 3, 4, 4, 5, 2, 6, 6]  连接两个或多个数组，返回一个新的数组

    # join()

    join() 方法将一个数组的所有元素连接成一个字符串并返回这个字符串

    参数： separator （可选）

    ​ 指定一个字符串来分隔数组的每个元素，转换为字符串， 如果缺省该值，数组元素用逗号(,)分隔，如果传入是空值（""）则所有元素之间都没有任何字符。

    const arr = [1,2,3]
let str1 = arr.join()
console.log(str1); // 1,2,3

let str2 = arr.join(':')
console.log(str2); // 1:2:3

let str3 = arr.join("")
console.log(str3); //123

    # push() 和 unshift()

    push() 方法将一个或多个元素添加到数组的末尾，并返回该数组的长度

    const arr = [1,2]
console.log( arr.push('3'));  // 3 
console.log(arr); // [1, 2, '3']

    2
    3

    unshift() 向数组的开头添加一个或多个元素，并返回该数组的长度

    const arr = [1,2]
console.log(arr.unshift('0')); // 3
console.log(arr); // ['0', 1, 2]

    # shift() 和 pop()

    shift() 从数组中删除第一个元素，如果数组为空则返回 undefined

    const arr = [3,2,3]
console.log(arr.shift()); // 3
console.log(arr); // [2,3]

    pop() 删除并返回数组最后一个元素，如果数组为空则返回 undefined

    const arr = [1,2,3]
console.log(arr.pop()); // 3
console.log(arr); // [1,2]

    # sort()

    sort() 方法用对数组的元素进行排序，并返回数组。默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列时构建的

    const arr = [2,4,8,0]

console.log(arr.sort());  // [0, 2, 4, 8]
console.log(arr); // [0, 2, 4, 8]

    # reverse()

    reverse() 方法将数组中元素的位置颠倒，并返回该数组 ，该方法会改变原数组

    const arr = [4,3,2,1]

console.log(arr.reverse()); // [1, 2, 3, 4]
console.log(arr); // [1, 2, 3, 4]

    # slice()

    slice(start,end) ** 方法返回一个新的数组，这个数组是由开始传入的 start 和 end 决定原数组的浅拷贝**，原始数组不会被改变

    const arr = [4,3,2,1]

console.log(arr.slice(1));  // [3,2,1]
console.log(arr.slice(1,3)); // [3,2]
console.log(arr.slice(1,-1)); // [3,2]
console.log(arr);  // [4, 3, 2, 1]

    # splice

    splice 更新数组 改变数组

    splice(start,deleteCount,item) 向/从数组中添加/删除项目，然后返回被删除的项目，返回含有被删除的元素的数组，若没有删除元素则返回一个空数组

    • start 必传 整数，规定添加/删除项目的位置，负数表示从数组结尾处规定位置
    • deleteCount 必传 要删除的项目数量。如果设置为 0，则不会删除项目
    • item 可选 向数组添加的新项目
    const arr = [4,3,2,1]

console.log(arr.splice(2,1)); //[2]
console.log(arr);  // [4,3,1]

    # indexOf() 和 lastIndexOf()

    两个方法都是返回要查找的项所在数组中首次出现的位置，没找到的话就返回-1。

    indexOf()

    从数组的开头开始向后寻找。

    let str = 'Blue Whale'
console.log(str.indexOf('z')); // -1
console.log(str.indexOf('e')); // 3

    lastIndexOf()

    从数组的末尾开始向前查找

    let str = 'Blue Whale'
console.log(str.lastIndexOf('e'));  // 9

    forEach() 遍历元素，并不会改变原数组中的值。

    const attr = ['a', 'b', 'c'];

let run = attr.forEach((item,index,array) => {
      console.log(item,index,array); // a 0 ['a', 'b', 'c']
});

    # filter()

    用来测试数组中每个元素的函数，返回true表示该元素通过测试，保留该元素，false 则不保留，filter()方法不会对空数组进行检测，不会改变原始数组

    const attr = [10,2,4,8,3,9]

let run = attr.filter((item,index,array)=>{
    return item>5

    # every()

    every() 方法用于检测数组所有元素是否都符合指定条件，数组中检测到有一个元素不满足，则整个表达式返回 false ，且剩余的元素不会再进行检测，所有元素都满足条件，则返回 true

    • 注意： every() 不会对空数组进行检测。
    • 注意： every() 不会改变原始数组。
    const attr = [10,2,4,8,3,9]
let str = attr.every((item,index,array)=>{
    return item>5
})
console.log(str); // false

    # some()

    some() 方法用于检测数组中的元素是否满足指定条件，如果有一个元素满足条件，则表达式返回true , 剩余的元素不会再执行检测，如果没有满足条件的元素，则返回false。

    • 注意： some() 不会对空数组进行检测
    • 注意： some() 不会改变原始数组
    const attr = [2,10,4,8,3,9]
let str = attr.some((item,index,array)=>{
    return item>3
})
console.log(str); // true
    1

    some 和 every 的区别

    • some() 方法，遍历数组每一项，其中一项为true，则返回 true
    • every() 方法，遍历数据每一项，全部为true，则返回 true

    # ruduce()

    ruduce() 方法接收一个函数作为累加器，数组每个值（从左到右）开始计算为一个值，返回计算的结果

    var arr = [1,2,3]

let sum = arr.reduce((total,currenValue,currentIndex)=>{
    console.log(total,currenValue,currentIndex);  // 第一次输出为  2 1 0
    return total+currenValue
},2)

console.log(sum); // 8

    # fill()

    fill()  方法用一个固定值填充一个数组中从起始索引到终止索引内的全部元素 ， 会改变原数组

    var arr = [1,2,3]
let str = arr.fill(6)

console.log(str); // [6, 6, 6]
console.log(arr);  // [6, 6, 6]

    # find()

    find() 为数组中的每个元素都调用一次函数执行。当数组中的元素在测试条件时返回 true 时, find() 返回符合条件的元素，之后的值不会再调用执行函数，如果没有符合条件的元素返回 undefined

    var ages = [3, 10, 18, 20];

let str = ages.find((item)=>{
   return item>21
})
console.log(str); // 10
